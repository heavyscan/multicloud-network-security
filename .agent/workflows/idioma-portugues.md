# Idioma Padrão: Português Brasileiro (pt-BR)

Esta diretriz se aplica a **todos os projetos** neste workspace.

## Regras

1. **Todo texto de saída** (respostas, descrições, explicações, comentários ao usuário) **deve ser escrito em Português do Brasil (pt-BR)**.
2. Nomes de variáveis, funções e código-fonte podem permanecer em inglês, conforme convenção técnica.
3. Mensagens de commit, descrições de pull requests e documentação voltada ao usuário devem ser em pt-BR.
4. Comentários inline no código podem ser em pt-BR quando forem explicativos para o usuário.
